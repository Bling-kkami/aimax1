## 2. 만들 AI 직원

직장인 AI 꿀팁 영상 기획 자동화 직원 — 모든 사무직이 겪는 반복 업무(문서정리, 엑셀, 보고서, 재고파악)를 AI로 빠르게 끝내는 팁들의 주제를 매주 발굴하고 각각 영상 기획서(제목, 핵심 포인트, 예시)를 자동 생성

## 3. 반복 업무와 해결할 문제

**핵심 문제**: 사무직 직장인들이 문서작업, 엑셀, 보고서 작성, 재고파악 같은 반복업무에 시간을 많이 쓰고 있음

**자동화 후 기대 효과**: 직장인들이 이 팁을 알면 주당 몇 시간의 업무 시간을 절약할 수 있음

**대상 사용자**: 김윤미 — 기타 / 직장인, 상세페이지디자이너,유튜버 준비중 업무 담당자

---

## 4. 입력값과 처리 흐름

Claude 또는 ChatGPT에게 '사무직 직장인들이 자주 하는 업무 목록'과 '각 업무를 AI로 단축하는 팁들'을 학습시킨 후, 매주 신규 주제 기획서를 자동 생성하도록 프롬프트 설계

## 5. 오늘 만들 MVP

**MVP 범위**: AI 직원이 사무직 공통 업무 기반 AI 꿀팁 주제 3~5개를 자동 발굴하고, 각 주제별로 영상 제목 + 핵심 포인트(스크립트) + 필요한 예시/화면을 담은 기획서를 한 세트로 완성하기

오늘은 위 범위 안에서 실제로 한 번 실행되는 결과물을 만드는 것이 목표입니다.
기능을 넓히기보다, 신청자가 직접 넣은 입력값 1개가 처리되어 결과 1개로 나오는 흐름을 먼저 완성합니다.

---

## 6. 첫 화면에 필요한 최소 요소

- 입력 영역 (텍스트 입력 또는 파일 업로드)
- 실행 버튼
- 결과 출력 영역
- 에러가 났을 때 다시 시도할 수 있는 안내

## 7. 오늘 만들지 않는 것

- 외부 배포 링크 생성

## 8. 개발 진행 기준

**Phase 1 — 오늘 MVP**
- 위 5번 MVP 범위만 구현
- 실제 샘플 입력 1개로 결과가 화면에 나오는지 확인
- 신청자가 결과물을 복사하거나 바로 사용할 수 있게 마무리

**Phase 2 이후 — 수업 후 확장**
- 저장 / 이력 관리
- 공유 / 이메일 전송
- 로그인 / 결제
- 외부 배포
- 다중 입력 처리

**항상 주의**
- API 키는 코드에 직접 쓰지 않기 — 환경변수(.env)로 관리
- 개인정보는 샘플 데이터로 대체해서 테스트
- GitHub Public 저장소에 .env 파일 올리지 않기
