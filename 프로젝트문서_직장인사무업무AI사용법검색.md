# 직장인 사무업무 AI 사용법 검색 웹앱 프로젝트 문서

> 회의용 정리 문서  
> 작성일: 2026년 5월 10일

---

## 1. 프로젝트 개요

**프로젝트명**  
직장인 사무업무 AI 사용법 검색

**한 줄 설명**  
비서실, 경영지원/총무, 영업지원, 온라인 쇼핑몰 운영, 상세페이지/마케팅 보조 실무자가 사무업무를 입력하거나 선택하면, AI를 활용해 해당 업무를 처리하는 방법을 쉽게 안내해주는 웹앱입니다.

**기획 의도**  
AI를 잘 모르는 실무자도 자신의 업무만 선택하면 AI 활용 방법을 바로 확인할 수 있게 하고, AI 활용에 익숙한 사람은 직접 업무를 입력해 더 구체적인 처리 방법을 얻을 수 있게 합니다.

---

## 2. 해결하려는 문제

사무실무자는 문서 정리, 보고서 작성, 회의록 요약, 이메일 작성, 엑셀 검수, 재고 정리, 고객 문의 답변 같은 반복 업무에 많은 시간을 씁니다.

하지만 AI를 활용하려고 해도 다음과 같은 어려움이 있습니다.

- 어떤 업무에 AI를 써야 할지 모름
- AI에게 어떻게 지시해야 할지 모름
- 결과가 맞는지 검수하는 방법을 모름
- 업무별로 어떤 자료를 준비해야 하는지 모름
- AI 활용법을 검색해도 실무에 바로 적용하기 어려움

이 웹앱은 사용자가 업무 상황을 선택하거나 입력하면, AI가 업무 처리 방법, 복사해서 쓸 수 있는 프롬프트, 결과물 예시, 검수 체크리스트를 제공합니다.

---

## 3. 주요 사용자

현재 MVP에서 우선 고려하는 사용자는 다음과 같습니다.

- 비서실 실무자
- 경영지원/총무 담당자
- 영업지원 담당자
- 온라인 쇼핑몰 운영 담당자
- 상세페이지/마케팅 보조 실무자

공통 특징:

- 반복 사무업무가 많음
- 빠르게 결과물을 만들어야 함
- AI를 잘 모르거나, 알아도 업무에 어떻게 적용할지 막막함
- 보고서, 표, 이메일, 체크리스트, 정리문 같은 실무 결과물이 필요함

---

## 4. 현재 웹앱 구조

### 앱 제목

**직장인 사무업무 AI 사용법 검색**

### 사용 방식

현재 앱은 두 가지 방식으로 사용할 수 있습니다.

**쉬운 선택**

- AI를 잘 모르는 사용자용
- 직무 탭에서 자신의 직무를 선택
- 자주 하는 업무 버튼을 누르면 입력창이 자동으로 채워짐
- 사용자는 `AI 직원 실행하기` 버튼만 누르면 됨

**직접 입력**

- AI 활용에 익숙한 사용자용
- 사용자가 직접 처리하고 싶은 업무를 자유롭게 입력
- 구체적인 업무 상황을 넣을수록 더 실무적인 결과 생성

---

## 5. 직무별 업무 구성

### 비서실

- 회의록 정리
- 대표 보고 브리핑
- 일정표 작성
- 이메일 초안
- 방문/의전 준비

### 경영지원/총무

- 사내 공지문 작성
- 비품/자산 관리
- 계약서 누락 검수
- 비용/영수증 정리
- 사내 규정 요약

### 영업지원

- 거래처 이메일
- 견적/발주 정리
- 고객 문의 답변
- 영업 보고서 초안
- 거래처별 현황 요약

### 온라인 쇼핑몰 운영

- 주문/재고 정리
- CS 답변 초안
- 상품 리뷰 요약
- 경쟁사 상품 조사
- 상품명/옵션 정리

### 상세페이지/마케팅 보조

- 상세페이지 구성안
- 상품 USP 정리
- 후킹 문구 만들기
- 광고 소재 아이디어
- 리뷰 기반 소구점

---

## 6. 생성되는 결과물

사용자가 업무를 입력하거나 선택하면 현재 앱은 다음과 같은 형태의 결과를 생성합니다.

- AI로 처리 가능 여부
- AI가 줄일 수 있는 업무와 사람이 확인해야 할 업무 구분
- 단계별 처리 방법
- 바로 복사해서 쓸 수 있는 AI 프롬프트
- 입력 자료 준비 방법
- 개인정보/회사 기밀 가리는 방법
- 결과물 예시
- 결과 검수 체크리스트
- 추가로 물어보면 좋은 질문
- 참고한 검색 기준 및 출처

결과는 화면에서 확인할 수 있고, 텍스트 파일로 다운로드하거나 PC의 `outputs` 폴더에 자동 저장됩니다.

- `.gitignore`로 `.env` 보호

## 8. 현재 MVP의 장점

- AI를 모르는 사용자도 버튼 선택만으로 시작 가능
- 직무별 업무 흐름이라 실제 실무자에게 설명하기 쉬움
- 단순 정보 검색이 아니라 업무 처리 방법까지 제공
- AI 프롬프트와 검수 체크리스트가 함께 제공되어 실무 적용성이 높음
- 상용화 시 직무별 템플릿 상품으로 확장 가능

---

## 9. 현재 한계

- 아직 로그인/회원 관리 없음
- 결과 저장 이력 관리 없음
- Word, Excel, PDF 다운로드 기능 없음
- 사용자가 실제 파일을 업로드해 처리하는 기능 없음
- 회사별 양식이나 내부 규정에 맞춘 커스터마이징은 아직 없음
- 결과 품질은 입력 내용과 Gemini 응답에 영향을 받음

## 12. 추천 포지셔닝

**추천 방향**  
AI를 잘 모르는 사무실무자도 업무 버튼만 누르면 사용할 수 있는 **직무별 AI 업무비서**로 포지셔닝하는 것이 좋습니다.

**추천 문구**  
사무업무를 입력하면, AI가 처리 방법과 실행 프롬프트를 알려주는 직장인 업무비서

**핵심 가치**  
사용자가 AI를 공부하지 않아도, 자신의 업무를 AI에게 시키는 방법을 알 수 있게 해줍니다.

## 13. 데모 시나리오 예시

회의에서 보여줄 수 있는 간단한 시연 흐름입니다.

1. 웹앱 접속
2. `쉬운 선택` 선택

6. `AI 직원 실행하기` 클릭
7. 결과에서 처리 방법, 프롬프트, 검수 체크리스트 확인
8. 결과 다운로드 또는 자동 저장 경로 확인

## 14. 결론

현재 웹앱은 단순한 AI 챗봇이 아니라, 직무별 사무업무를 AI로 처리하도록 안내하는 실무형 도구입니다.

초기에는 비서실, 경영지원/총무, 영업지원, 온라인 쇼핑몰 운영, 상세페이지/마케팅 보조 실무자를 대상으로 업무 템플릿을 제공하고, 이후 파일 업로드, Word/Excel 다운로드, 이력 관리, 팀 사용 기능으로 확장할 수 있습니다.

상용화를 위해서는 먼저 특정 직무군 하나를 정해 실제 사용자에게 테스트하고, 가장 많이 쓰는 업무 템플릿과 결과물 형식을 검증하는 것이 중요합니다.
